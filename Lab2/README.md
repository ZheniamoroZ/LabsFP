# crawler

Crawl web-pages for links. Use found links to crawl further.

## Usage

	$ lein run [file-with-urls] [depth-of-crawling]
    $ java -jar crawler-0.1.0-standalone.jar [file-with-urls] [depth-of-crawling]

## Examples

	$ lein run urls.txt 1
